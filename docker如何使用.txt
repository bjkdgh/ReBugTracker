如何使用
构建镜像: 在你的项目根目录（Dockerfile 和 .dockerignore 所在的目录）打开终端，运行以下命令：

docker build -t rebugtracker-app .

-t 参数为你的镜像命名，方便后续使用。

运行容器: 使用 docker run 命令来启动你的应用容器。注意，这里我们通过 -e 参数来安全地注入数据库连接字符串。

docker run -d \
  --name my-rebugtracker \
  -p 5000:5000 \
  -e SQLALCHEMY_DATABASE_URI="postgresql://your_user:your_password@your_db_host/your_dbname" \
  rebugtracker-app

-d: 后台运行容器。
--name: 给容器一个友好的名字。
-p 5000:5000: 将主机的 5000 端口映射到容器的 5000 端口。
-e: （关键） 设置环境变量，将你的实际数据库连接信息传递给容器。
现在，你的应用就已经基于一个安全、高效、标准的 Docker 镜像运行起来了！
